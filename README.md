# Better Item Frame Maps

## English

Better Item Frame Maps is an addon for Better Than Wolves Community Edition 3.0.0 (Minecraft 1.6.4). It significantly improves how maps are rendered inside Item Frames.

### Features
*   **Seamless Maps:** Maps displayed in Item Frames cover the full block face (1x1).
*   **No Borders:** The wooden frame and the paper background are hidden.
*   **Continuous Maps:** Placing Item Frames next to each other creates a large, continuous map wall without gaps.

### Configuration
The mod generates a configuration file at `.minecraft/config/BetterItemFrameMaps.properties`.

*   **enableRotation** (default: `true`)
    *   `true`: You can rotate the map by right-clicking the Item Frame.
    *   `false`: Maps are locked in a fixed "North-up" orientation.
*   **enableDebugLogs** (default: `true`)
    *   Enables logs.

### [Downloads](https://github.com/zamroczynski/better-item-frame-maps/releases)

### Installation
Place the downloaded `.jar` file into your `.minecraft/mods` folder.

## Polski

Better Item Frame Maps to addon do modyfikacji Better Than Wolves Community Edition 3.0.0 (Minecraft 1.6.4). W znaczący sposób ulepsza on renderowanie map umieszczonych w ramkach na przedmioty.

### Funkcje
*   **Pełny rozmiar:** Mapy w ramkach zajmują całą powierzchnię bloku (1x1).
*   **Brak ramek:** Drewniana ramka oraz papierowe tło mapy są ukryte.
*   **Ciągłość map:** Umieszczenie ramek obok siebie tworzy jedną, dużą mapę na ścianie bez widocznych przerw.

### Konfiguracja
Modyfikacja generuje plik konfiguracyjny w `.minecraft/config/BetterItemFrameMaps.properties`.

*   **enableRotation** (domyślnie: `true`)
    *   `true`: Można obracać mapę klikając prawym przyciskiem myszy na ramkę.
    *   `false`: Mapy są zablokowane w stałej pozycji (północ na górze).
*   **enableDebugLogs** (domyślnie: `true`)
    *   Włącza logi.

### [Pobieranie](https://github.com/zamroczynski/better-item-frame-maps/releases)

### Instalacja
Umieść pobrany plik `.jar` w swoim folderze `.minecraft/mods`.
